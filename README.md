# pusher.io gem

This is very basic implementation of subset of pusherapp.com api on top of socket.io.

## Installation

Add this line to your application's Gemfile:

    gem 'pusher.io'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pusher.io
